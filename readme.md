## HANG MAN GAME ##

Hang man game in python 3 derived from a training. 

## This knowledge was applied: ##
    - Using the random library to scroll through a list and return a position. 
    - Use of Python native file reading.
    - Utilization of list comprehension.
    - Implemented the "Enumerate" function.
    - Implemented interpolation in Python.

## How to install and play. ##

    1. Install Python3 on your machine.
    2. Download the repository and open the folder in the terminal (linux)
    3. Use the command "python3 ./hangman.py"
    4. At each error an attempt is lost, when it reaches zero you lose.
    5. If you get the word right, the game ends and your score is displayed (0 to 100)
